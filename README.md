# Sanskrit AI

This project aims to create an AI-powered chatbot similar to ChatGPT and Gemini, with the following features:

1. **Chat-like Interface**: Users can interact with the system using natural language, and the system will understand and respond appropriately using machine learning.
2. **Deployment Ready**: The project will be configured for deployment on Render.
3. **MongoDB Integration**: User data and conversation history will be stored in MongoDB, with a dashboard to view previous conversations.
4. **Admin Subdomain**: An admin interface for managing the system.
5. **Modern UI**: A sleek, glassy design inspired by Microsoft 365 dashboards.

## Features
- **Machine Learning Backend**: Understands user intent and generates responses.
- **MongoDB Integration**: Stores user data securely.
- **Admin Dashboard**: Provides administrative control.
- **Modern UI**: User-friendly and visually appealing.

## Setup Instructions
1. Clone the repository.
2. Install dependencies.
3. Configure MongoDB connection.
4. Deploy to Render.

## Folder Structure
- `backend/`: Contains the machine learning backend.
- `frontend/`: Contains the user interface.
- `admin/`: Contains the admin interface.

## Deployment
This project is configured for deployment on Render. Ensure all environment variables are set correctly before deploying.
